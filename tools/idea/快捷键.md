## IDEA常用快捷键



### 窗口显示

| 功能                 | WIN     | MAC         |
| -------------------- | ------- | ----------- |
| 显示/隐藏message窗口 | ALT + 0 | Command + 0 |
| 显示/隐藏项目        | ALT + 1 | Command + 1 |
| 显示/隐藏收藏        | ALT + 2 | Command + 2 |
| 显示/隐藏TODO        | ALT + 2 | Command + 6 |
